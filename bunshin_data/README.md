# bunshin_data — 分身の教科書づくり(フェーズ2)

自分のデータを「AIが学習できる形式」に変換する工場。

## フォルダの使い方

```
bunshin_data/
├── make_dataset.py   ← 変換プログラム(python make_dataset.py で実行)
├── raw/              ← ここに素材を入れる
│   ├── line/         ← LINEのトーク履歴(.txt)を入れる
│   ├── ai/           ← ChatGPT/Claudeの会話エクスポート(.json)を入れる
│   ├── style/        ← 自分の文章(.txt)を入れる(日記の文字起こしなど)
│   └── diary_photos/ ← 紙の日記の写真を入れる(Claudeが文字起こしする)
└── output/           ← 変換結果が自動で出てくる(dataset.jsonl など)
```

## 素材の集め方

### LINEのトーク履歴(いちばん大事!会話ペアの宝庫)

1. スマホのLINEで、よく話す友達とのトークを開く
2. 右上の「≡」メニュー → **設定** → **トーク履歴を送信**
   (Androidは「その他」→「トーク履歴をバックアップ」の場合も。
   機種やバージョンでメニュー名が少し違うが「トーク履歴を送信/バックアップ」を探す)
3. 出てきた .txt ファイルを自分にメールするかGoogleドライブ等でPCに送る
4. `raw/line/` フォルダに入れる(何個でもOK。多いほど良い)

※グループトークより1対1のトークの方が「あなたの返事」がきれいに取れる。

### 紙の日記(文体の教科書)

スマホで写真を撮って `raw/diary_photos/` に入れて、Claudeに
「日記の写真を入れたよ」と伝えるだけ。Claudeが読んで文字起こしして
`raw/style/` にテキストとして保存してくれる。

### 生成AIとの会話履歴(あなたの書き方・考え方の見本)

エクスポート方法(どれもzipがメールで届く or ダウンロードできる):

- **ChatGPT**: 設定 → データコントロール → **データをエクスポート** → 届いたzipの中の
  `conversations.json` を `raw/ai/` に入れる(名前は `chatgpt.json` などに変えてOK)
- **Claude**: claude.ai の Settings → Privacy → **Export data** → zipの中の
  `conversations.json` を `raw/ai/` に入れる(`claude.json` などに変えてOK)
- **Gemini**: [Google Takeout](https://takeout.google.com) で「Gemini」だけ選んで
  エクスポート。形式が特殊なので、届いたらClaudeに渡せば対応する

ChatGPT形式かClaude形式かはプログラムが自動判別する。学習に使われるのは
**自分が書いたメッセージだけ**(AIの返答を分身の口調にはしない)。
コード貼り付けや「続けて」のような短い指示は自動で除外される。

### Discord(少し手間なので後回しでOK)

ユーザー設定 → プライバシー・安全 → 「すべてのデータをリクエスト」で
公式にエクスポートできるが、届くまで数日〜数週間かかる。先にリクエスト
だけ出しておくのがおすすめ。

## 変換の実行

```
cd bunshin_data
python make_dataset.py
```

- 初回は `make_dataset.py` の上の方にある `MY_NAME = "いぶき"` を
  自分のLINE表示名に書き換える(自分の発言を見分けるために必要)
- `output/dataset.jsonl` = 会話ペア集。フェーズ3(Colabでのファインチューニング)で使う
- `output/style_corpus.txt` = 自分の文章の詰め合わせ。これを `mini_gpt/data/input.txt` に
  コピーすれば、フェーズ1の豆GPTを「自分の文体」で学習し直すこともできる

## プライバシーについて

- 変換はすべてこのPCの中だけで完結する(どこにも送信されない)
- フェーズ3でColab(Google)にアップロードする前に、友達の名前などを
  伏せ字にするチェックを一緒にやる
- 相手の発言も学習データに含まれるので、公開したり配布したりはしないこと

## 目安

| ペア数 | 効果 |
|---|---|
| 〜500組 | 口調がちょっと似てくる |
| 500〜2000組 | 返事のクセ・定番の返しがかなり再現される |
| 2000組〜 | 友達が見たら「お前やん」と言うレベルを狙える |
